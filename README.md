The Lax PSR2 coding standard.
====

It is PHP coding standard based SPR2.

* 2-spaces indent.

## Requirements

* The PHP Code Sniffer

## Installation

```bash
$ cd standards
$ git clone https://github.com/satoruk/LaxPSR2.git
$ phpcs --config-set default_standard ./LaxPSR2
```
